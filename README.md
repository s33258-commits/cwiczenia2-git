\# cwiczenia2-git



Prosty projekt konsolowy w .NET do ćwiczeń z Git i GitHub.



\## Merge bez fast-forward



Tym razem merge nie był fast-forward, ponieważ po utworzeniu gałęzi `feature-max`

na gałęzi `main` został dodany osobny commit. Historia gałęzi rozeszła się,

dlatego Git musiał utworzyć osobny commit scalający.
